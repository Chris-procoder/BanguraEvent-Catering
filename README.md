# Bangura Event & Catering - Culinary Arts & Hair Styling Website

A professional website showcasing Bangura Event & Catering's dual expertise in culinary arts and hair styling services. This project was recreated and adapted from the Mett-Apperals tailoring website design.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Dual Service Focus** - Separate booking systems for catering and hair styling services
- **Interactive Gallery** - Tabbed gallery showcasing culinary work and hair styling portfolio
- **AI Chatbot** - 24/7 customer support with intelligent responses about services
- **Hero Slider** - Dynamic slideshow highlighting different service aspects
- **Smooth Animations** - Modern CSS animations and transitions
- **Contact Forms** - Professional booking forms with validation
- **Testimonials** - Client feedback slider with auto-advance
- **Lightbox Gallery** - Click-to-expand image viewing

## Setup Instructions

1. **Clone or Download** the project files to your local machine
2. **Open Terminal/Command Prompt** and navigate to the project directory
3. **Start Local Server**:
   ```bash
   python -m http.server 8000
   ```
4. **Open Browser** and visit: `http://localhost:8000`
